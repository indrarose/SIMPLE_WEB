# SIMPLE_WEB

Configuration File :
conf/app.properties

MYSQL auto create database and tables if doesnt exist;

EDITOR : Intellij IDEA 

CLONE TO LOCAL : 
1. git clone
2. cd [projectname]
3. mvn compile
4. mvn package

RUNNING (Mac OS) :
java -cp "target/lib/*:target/SIMPLE_WEB-1.0-SNAPSHOT.jar" net.rose.simple.Main

OPEN WEB:
http://localhost:8000/
